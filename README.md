# Whack-a-Mole

A classic, browser-based arcade game built with Vanilla JavaScript, HTML, and CSS. Test your reflexes by whacking the Monty Moles to score points, but watch out for the Piranha Plants! 

## 🎮 Features

* **Dynamic Entity Spawning:** Moles and Piranha Plants appear randomly across a 12-pipe grid.
* **Score Tracking:** Real-time point accumulation for every successful mole hit.
* **Hazard Mechanics:** Instantly triggers a "Game Over" state if a Piranha Plant is clicked.
* **Themed UI:** Utilizes custom Mario-themed visual assets for an engaging retro aesthetic.

## 🛠️ Technologies Used

* **HTML5:** For structuring the game board and UI elements.
* **CSS3:** For styling the grid layout, pipes, and visual presentation.
* **JavaScript (Vanilla):** For DOM manipulation, randomized spawning logic, and event handling.
